# Прайс-листы в формате JSON

Этот каталог содержит данные о ценах для компонента `Prices.tsx` в структурированном формате JSON.

## Файлы

- **master_prices.json** - цены для категории «Мастер» (74 услуги)
- **top_master_prices.json** - цены для категории «Топ-Мастер» (62 услуги)

## Структура данных

```json
{
  "Название категории": [
    {
      "service": "Название услуги",
      "price": 1000,
      "prefix": "от "
    }
  ]
}
```

### Поля

- **service** (string, обязательное) - название услуги
- **price** (number, обязательное) - цена в рублях (число без символа ₽)
- **prefix** (string, опциональное) - префикс для цены:
  - `"от "` - для цен с диапазоном (результат: "от 100₽")
  - `"+"` - для доплат (результат: "+500₽")

## Категории услуг

### Для "Мастер":
1. Маникюр и уход за руками (21 услуга)
2. Педикюр и уход за ногами (24 услуги)
3. Дополнительные услуги (29 услуг)

### Для "Топ-Мастер":
1. Маникюр и уход за руками (15 услуг)
2. Педикюр и уход за ногами (17 услуг)
3. Дополнительные услуги (30 услуг)

## Использование

Данные автоматически импортируются в компонент `src/components/Prices/Prices.tsx`:

```typescript
import masterPricesData from "../../../prices_json/master_prices.json";
import topMasterPricesData from "../../../prices_json/top_master_prices.json";
```

## Редактирование цен

Чтобы обновить цены:

1. Отредактируйте соответствующий JSON файл
2. Сохраните изменения
3. Компонент автоматически отобразит новые данные при следующей сборке

**Внимание:** При редактировании убедитесь, что JSON остается валидным!
